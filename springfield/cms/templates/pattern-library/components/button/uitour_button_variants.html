These buttons are only functional within Firefox desktop.

Local testing:

the UI tour functionality only works on mozilla domains, so to make it work locally (on http://localhost:8000/), you will need to:

  1. open Firefox and go to about:config
  2. change browser.uitour.requireSecure to false
  3. change browser.uitour.testingOrigins to http://localhost:8000/
