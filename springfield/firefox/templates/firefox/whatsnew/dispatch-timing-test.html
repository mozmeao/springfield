UITour timing test

This page measures how long Mozilla.UITour.ping() and two getConfiguration() calls take on your browser. Results are sent to Springfield metrics and are also shown below.

No browser-state values are transmitted — only timings and success/failure flags. See the prototype code for the full payload.

Not seeing measurements?

UITour is only exposed by Firefox on allowlisted origins over HTTPS. For local testing on non-secure origins, set the following prefs in about:config:

  • browser.uitour.requireSecure → false
  • browser.uitour.testingOrigins → your local origin (e.g. http://localhost:8000/)

On non-Firefox browsers, the page will report no_uitour as its outcome.
